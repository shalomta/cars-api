Cars API Dcumentaion

Cars Routes

Method URL Body Description
GET /cars Get a list of all the cars, will present 10 results per page
GET /cars/?s=compact search for a car or a car description, will present 10 results per page
GET /cars/?page=2 Get a list of all the cars, will present 10 results in page 2
GET /cars/cat/chevrolet Get a list of all the cars in chevrolet category, will present 10 results per page
GET /cars/prices/?min=40000&max=55000 Get a list of all the cars priced between 40,000 and 55000, will present 10 results per page
POST /cars 
{
    "name": "Ford Explorer",
    "info": "Info about the car",
    "category": "Ford",
    "img_url": "image.jpg",
    "price": 67300
}
Add a new car to the data base, You need a token for this query
PUT /cars/1 
{
    "name": "Ford Explorer",
    "info": "Info about the car",
    "category": "Ford",
    "img_url": "image.jpg",
    "price": 66300
}
Update a car in the data base, You need a token for this query
DELETE /cars/1 Delete a car from the data base, You need a token for this query
